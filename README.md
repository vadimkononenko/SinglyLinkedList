# Singly Linked List

Demonstrates work with typed singly linked list including UNIT-tests

## How to run?

Clone the repository:
```
$ git clone https://github.com/vadimkononenko/SinglyLinkedList 
```

Install Node.js and npm

## How to use?

Download all dependencies:

```
$ npm install
```

To run tests:

```
$ npm run test
```
